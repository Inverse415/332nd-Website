Servers

The 332nd has 4 deticated servers that unit members can play on.


Server 1: Operations and Main Operations

Server one hosts our main operations and is open for members to Zeus thier own missions.

Server 2: Training and Secondary Operations

Server 2 is primarily used for training, but it can also be used by members to run ops if server one is being used by someone else.

Server 3: Liberation

We host a liberation server that members can go play on at any time. The Mod list for this is updated regularly with new content.

Server 4: Event Server

The event server rotates out different events every couple of months. You can learn more about event management projects on the projects and events page.

Minecraft Server

We have our own minecraft server that is open to unit guests and members. This is a modded server.

Game of the Month Server

Unit Members Vinnie and Jager partnered with event management to create a game of the month server! This server changes games monthly and allows members to vote for and play games outside of arma on a dedicated server.
